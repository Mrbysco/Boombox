# Boombox #

## About ##
Adds a boombox that can play radio stations.
The boombox can be right-clicked to reveal a list of radio channels. Upon clicking a channel it will tune into that channel. Holding shift while clicking a channel will favorite that channel, moving it to the top.

This mod is a standalone and updated version of the [Ad Astra](https://www.curseforge.com/minecraft/mc-mods/ad-astra) Radio block.
